# Jira REST SDK Icons

This directory contains the official icons for the Jira REST SDK project.

## Available Icons

### Main Icon
- **jira-sdk-icon.svg** - Main project icon (128x128)
  - Features Jira's signature blue gradient
  - Includes ticket/issue representation
  - SDK gear indicator
  - REST API connection dots

### Size Variants
- **jira-sdk-icon-16.svg** - Small icon (16x16) - Simplified version for small displays
- **jira-sdk-icon-32.svg** - Medium icon (32x32) - Good for toolbars and buttons
- **jira-sdk-icon-64.svg** - Large icon (64x64) - Detailed version for larger displays

### Web Usage
- **favicon.svg** - Favicon version (16x16) - Optimized for browser tabs and bookmarks

## Design Elements

### Colors
- **Primary Blue**: `#0052CC` to `#2684FF` (Jira brand colors)
- **Accent Orange**: `#FF5630` to `#FF8F73` (SDK/gear indicator)
- **White**: Used for contrast and readability

### Symbolism
- **Rounded rectangles**: Represent Jira tickets/issues
- **Gear icon**: Indicates SDK/development tools
- **Connection dots**: Represent REST API connectivity
- **Circular background**: Modern, clean design approach

## Usage Guidelines

### In Documentation
```markdown
![Jira REST SDK](assets/icons/jira-sdk-icon.svg)
```

### In HTML
```html
<img src="assets/icons/jira-sdk-icon.svg" alt="Jira REST SDK" width="64" height="64">
```

### As Favicon
```html
<link rel="icon" type="image/svg+xml" href="assets/icons/favicon.svg">
```

## File Formats
All icons are provided in SVG format for:
- Scalability without quality loss
- Small file sizes
- Easy customization
- Wide browser support

To convert to other formats (PNG, ICO), use online converters or tools like ImageMagick.
